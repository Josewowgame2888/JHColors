# JHColors
add colors in the console Node.js

# Colors 

- `ESCAPE`
- `BLACK`
- `DARK_BLUE`
- `DARK_GREEN`
- `DARK_AQUA`
- `DARK_RED`
- `DARK_PURPLE`
- `GOLD`
- `GRAY`
- `DARK_GRAY`
- `BLUE`
- `GREEN`
- `AQUA`
- `RED`
- `LIGHT_PURPLE`
- `YELLOW`
- `WHITE`
- `OBFUSCATED`
- `BOLD`
- `UNDERLINE`
- `ITALIC`
- `RESET`
# Example
```
const Colors = require('jhcolors');

console.log(Colors.RED+'Hello world!'+Colors.RESET);
```
